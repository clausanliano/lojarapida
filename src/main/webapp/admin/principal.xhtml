Bem Vindo ao módulo administratrivo da Loja Rápida

A loja rápida foi desenvolvida durante o curso de desenvolvimento de web com java do SENAC



  • Efetue todas as operações com categorias como adicionar uma nova categoria, alterar ou excluri uma categoria existente.
